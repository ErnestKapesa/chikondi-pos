# Getting Started with Chikondi POS

Welcome! This guide will help you get Chikondi POS up and running in minutes.

## ðŸ“‹ Prerequisites

Before you begin, make sure you have:

- **Node.js 18+** installed ([Download here](https://nodejs.org/))
- **npm** (comes with Node.js)
- A modern web browser (Chrome, Firefox, Safari, or Edge)
- A code editor (VS Code recommended)

Check your versions:
```bash
node --version  # Should be v18.0.0 or higher
npm --version   # Should be v8.0.0 or higher
```

### Step 1: Install Dependencies

```bash
# Install frontend dependencies
npm install
```

This will install all required packages including React, Vite, TailwindCSS, and more.

### Step 2: Start Development Server

```bash
npm run dev
```

You should see:
```

### Step 3: Open in Browser

1. Open your browser
2. Go to: **http://localhost:5173**
3. You'll see the Chikondi POS setup screen

### Step 4: Create Your Account

1. **Shop Name**: Enter your business name (e.g., "My Shop")
2. **Create PIN**: Enter a 4-6 digit PIN (e.g., 1234)
3. **Confirm PIN**: Re-enter the same PIN
4. Click **Setup Account**
